Office Color Auto-Selection Test

Test Office Selection with Auto Color

Auto-selected based on office

How it works:

  1. Select an office from the dropdown
  2. The color picker automatically updates to match the office's color
  3. A color indicator appears next to the office dropdown
  4. Users can still manually change the color if needed
  5. When no office is selected, it defaults to blue (#3B82F6)

✅ Features Implemented:

  • Office data stored with colors in JavaScript
  • Event listener on office dropdown for color updates
  • Visual color indicator in office dropdown
  • Automatic color picker synchronization
  • Default color fallback when no office selected
  • Color indicator hidden/shown based on selection
